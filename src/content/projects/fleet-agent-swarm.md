Fleet Mode explores what happens when you stop thinking about AI coding as a single-agent conversation and start treating it as a swarm. Instead of one Copilot session grinding through tasks sequentially, Fleet deploys multiple agents in parallel — each working on a different part of the problem, coordinated through a shared context layer.

The landing page itself is a premium React + Vite project built with Framer Motion animations, a dark cinematic design language, and a full responsive layout. It showcases the Fleet Mode concept through animated terminal mockups, a simulated agent dashboard, and a three-phase architecture walkthrough. The design follows the Linear/Vercel-inspired aesthetic: minimal chrome, generous whitespace, scroll-triggered reveals with reduced-motion support.

Beyond the marketing surface, the Fleet concept connects directly to the broader agent orchestration work in [Agent Council](../agent-council/) and [Copilot CLI Harness](../copilot-cli-harness/). Where the Harness provides persistence and TDD structure for a single agent, and the Council coordinates multiple models for decision-making, Fleet Mode extends the pattern to parallel task execution across an entire codebase.
